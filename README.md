## Ejecutar Localmente

Clonar el proyecto

```bash
  git clone https://github.com/anndrees/PLANNING-AND-WORKING
```

Ir al directorio del proyecto

```bash
  cd my-project
```

Instalar dependencias

```bash
  npm install
```

Iniciar el servidor

```bash
  npm run start
```
